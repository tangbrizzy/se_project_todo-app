# Simple Todo App

Project 7 involved creating a simple application that allow an user to make a To-Do list to keep track and organize their daily activities.

## Functionality

The app involved using functions and validation methods in order to work efftively. Also, only valid input will allow the user to add additional task to their list and delete any tasks that are complete.

## Technology

This project includes using import/exporting between files. Adding new form that required the user to input valid data and adding a due date to individual task if needed. Also, if the input is invalid the error message will displayed and disabled the submit button.
